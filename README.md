# Phase 4 Project — NLP Twitter Sentiment Analysis

## Overview

This project analyzes tweets about Apple and Google products (CrowdFlower SXSW Twitter dataset) to classify sentiment as **Positive**, **Negative**, or **No emotion** toward a brand/product. The goal is to help marketing and product teams automatically monitor brand perception on social media instead of manually reading thousands of tweets.

## Project Summary

- **Business Problem:** Can tweet sentiment toward Apple/Google products be classified automatically to support brand monitoring?
- **Data Preparation:** Cleaned tweet text (lowercasing, removing URLs/mentions/hashtags/punctuation), then vectorized with TF-IDF (unigrams + bigrams).
- **Modeling:** Compared three models — Multinomial Naive Bayes (baseline), Logistic Regression, and Linear SVC — using a stratified train/test split.
- **Evaluation:** Weighted F1-score was used as the primary metric due to class imbalance. **Logistic Regression** was selected as the final model for its strong performance and interpretability.
- **Explainability:** LIME was used to show which words drive individual predictions, and global feature importances were plotted per sentiment class.
- **Recommendation:** Use the model as a first-pass triage tool to flag tweets (especially negative ones) for human review, and continue collecting negative-sentiment examples to improve minority-class performance.
